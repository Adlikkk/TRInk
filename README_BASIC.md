# TRInk Basic

TRInk Basic is the lightweight local edition of TRInk.

## Scope

- Select / Move
- Pen
- Highlighter
- Line
- Arrow
- Rectangle
- Eraser
- Undo / Redo / Clear
- Settings
- Quit

## Behavior

- Local only
- No account required
- No telemetry by default
- No broker integration
- No timer, trading tools, or price-action workflow in the Basic UI
- Basic toolbar stays fixed to Basic tools instead of Trading favorites

## Toolbar

- Drag handle and logo stay visible without overlapping controls
- Fixed-width tool buttons for `Select`, `Pen`, `Highlighter`, `Line`, `Arrow`, `Rectangle`, and `Eraser`
- `Undo`, `Redo`, `Clear`, `Settings`, and `Quit` stay visible

## Settings

- Tabs: `General`, `Tools`, `Keybinds`, `Appearance`, `About`
- No Timer, Trading, Price Action, or session/export sections in the Basic UI

## Commands

- `pnpm prepare:edition:basic`
- `pnpm dev:basic`
- `pnpm build:basic`
- `pnpm tauri:build:basic`
- `pnpm release:basic`
