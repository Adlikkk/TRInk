# TRInk Beta Summary

TRInk is a local TradeReality annotation overlay for Windows.

Current beta highlights:

- compact toolbar and grouped tool palette
- local save/load sessions
- local PNG and JSON annotation export
- editable objects with ordering and locking
- manual timer with neutral countdown wording
- custom keybinds with conflict handling
- local-only workflow with no in-app login, no telemetry by default, and no broker integration

Current beta limits:

- Windows-only distribution
- unsigned installer during the beta phase
- no cloud sync
- no broker/platform connection
