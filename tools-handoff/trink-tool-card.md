# TRInk Tool Card

TRInk, short for TradeReality Ink, is a free bonus tool for registered TradeReality users.

- Category: Tools
- Platform: Windows
- Price: Free
- Access: Registered users
- Description: A local screen annotation overlay for chart markup, trade review, and trading education.
- Status: Beta

Safety summary:

- No broker integration
- No trading automation
- No financial advice
- No telemetry by default
- Local sessions and exports only

TRInk runs locally on Windows and does not require login inside the desktop app.
