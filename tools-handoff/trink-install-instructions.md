# TRInk Install Instructions

1. Download `TradeReality Ink_0.3.11_x64-setup.exe` from the TradeReality Tools area.
2. Run the installer and complete the normal Windows install flow.
3. Launch `TradeReality Ink` from the Start Menu.
4. On first run, review the short welcome notice and continue.
5. Use the compact toolbar to draw, review, explain, and export local annotations.

Notes:

- TRInk is a local-only desktop overlay.
- It does not require login inside the desktop app.
- It does not connect to brokers or automate trades.
- Beta installers are currently unsigned, so Windows SmartScreen warnings are expected.
