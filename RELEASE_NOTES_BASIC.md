# TRInk Basic Release Notes

## v0.1.0

First Basic release candidate / first Basic release build.
Windows beta.
No account required.
No telemetry by default.
No trading tools.
Text tool deferred to v0.1.1.

- Introduced the shared two-edition architecture
- Added the `basic` build flavor
- Reduced the Basic UI to local screen annotation tools only
- Added a dedicated straight `Line` tool for Basic markup
- Simplified the Basic toolbar to fixed Basic tools and core actions only
- Locked Basic startup behavior to `Select / Move` plus click-through
- Split Basic settings into `General`, `Tools`, `Keybinds`, `Appearance`, and `About`
- Removed Trading-only palette sections, timer UI, and session/export controls from the Basic UI
